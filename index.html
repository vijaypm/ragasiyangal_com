Ragasiyangal
(ruh-guh-SEE-yung-ul)
is a free and open-source password manager.

  • Secure and auditable.
  • No remote server connections.
  • Totally free (no strings attached).
  • Get Started

Downloads

You can download pre-built binaries for Linux or Mac OS X by clicking on the buttons below. If you are paranoid like me and want to audit the code or build from source, you can get the source from GitHub.

  • Mac

Standard Functions

  • New - Create a new file
  • Open - Open an existing encrypted file
  • Save - Encrypt and Save changes to a New or Existing file
  • Exit - Quit the app

Protect any CSV file

  • Import any comma-separated (csv) file and convert to a secure encrypted file.
  • File can have an arbitrary number of columns.
  • The only requirement is for the file to have a header in the first row of the file.

Visual Feedback

Ragasiyangal was designed to not only be secure but also easy to use. The app relies heavily on vivid colors to guide the user.

What does 'Ragasiyangal' mean?

Ragasiyangal means 'secrets' in Tamil.Tamil is one of the few oldest surviving classical languages (as designated by UNESCO) in the world and is spoken by over 75 million people around the world.
